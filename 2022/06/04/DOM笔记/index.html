获取元素：document.querySelector（''）

获取多个元素：document.querySelectorAll（''） 得到的是一个伪数组，需要遍历(for)才获得

获取元素其他方法：

根据id获取元素 ：document.getElementById(' ')

根据标签获取元素 ：document.getElementByTagName(' ')

根据类名获取元素：document.getElementByClassName(' ')

修改或写入内容： 元素.innerHtml = ''
修改元素的值 ：例如：pic.src = './images/'
利用style 操作css ：例如 ： box.style.backgroundColor = 'red' 注意小驼峰
利用className操作css ：例如 ：div.className = ’active‘
利用classList操作css ：
追加一个类 ：元素.classList.add('类名')
删除一个类： 元素.classList.remove('类名')

切换一个类： 元素.classList.toggle('类名')

设置/修改表单元素属性
例如： 表单.value = '用户名' 表单.type = 'passwor'

开启定时器： setInterval(函数，间隔时间)
关闭定时器：let 变量名 = setInterval(函数，间隔时间) clearInterval(变量名)

事件监听 : 元素.addEventListener('事件'，要执行的过程) \*事件类型要加引号*

例如：btn.addEventListener('click',fuction() { })

事件类型
  • 鼠标触发 ：click 鼠标点击 mouseenter 鼠标经过 mouseleave 鼠标离开
  • 表单获得光标： focus 获得焦点 blur 失去焦点
  • 键盘触发 ： keydown 键盘按下触发 keyup 键盘抬起触发
  • 表单输入触发：input 用户输入事件

查找节点

父节点查找：子元素.parentNode 例如：son.parentNode.style.display = 'none'

子节点查找：父元素.children (返回的是一个伪数组)

下一个兄弟节点：nextElementSibling 属性

上一个兄弟节点：previousElementSibling 属性

增加节点：document.createElement('标签名')

插入到父元素的最后一个子元素： 父元素.appendChild(要插入的元素)

插入到父元素中某个子元素的前面：父元素.insertBefore(要插入的元素，在哪个元素前面)

克隆节点 ： 元素.cloneNode(布尔值) (true,则代表克隆时包含节点一起克隆) (false默认)

删除节点 ：父元素.removeChild(要删除的元素)

getFullYear() 获得年份 获取四位年份

getSeconds() 获取秒 取值为 0 ~ 59

事件绑定回调函数：元素.addEventListener('click',fuction(e)) { }

type 获取当前的事件类型

clientX/clientY 获取光标相对于浏览器可见窗口左上角的位置

offsetX/offsetY 获取光标相对于当前DOM元素左上角的位置

key 用户按下的键盘键的值 现在不提倡使用keyCode

阻止事件流动 事件对象.stopPropagation() 此方法可以阻断事件流动传播，不光在冒泡阶段有效，捕获阶段也有效
